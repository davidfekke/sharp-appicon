# Usage
<!-- usage -->
```sh-session
$ npm install -g sharp-appicon
$ appicon COMMAND
running command...
$ appicon (-v|--version|version)
sharp-appicon/0.2.1 darwin-x64 node-v14.15.0
$ appicon --help [COMMAND]
USAGE
  $ appicon COMMAND
...
```
